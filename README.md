This is a **fun login page** made with **HTML, CSS, and JavaScript**.  
It has cool animations, light & dark modes, stars, sun & moon, and 3D buttons. Works on mobile and desktop.

---

## Features

- **Light Mode:** Sun and login box appear slowly  
- **Dark Mode:** Background turns black, stars appear, moon slides in  
- **3D Buttons:** Buttons and inputs have hover effects  
- **Stars Animation:** Twinkling stars in dark mode  
- **Responsive:** Works on mobile, tablet, and desktop
